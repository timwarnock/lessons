How do we do this?

We do this with a programming language known as JavaScript (or ECMAScript).

JavaScript can be used for simple interactivity of a webpage (like the above kittens), or for creating advanced web applications.


Let's take a look

You can always view the source of this or any webpage to investigate the code yourself, but let's walk through this carefully.

Within the <head> of this page, I added the following:

<script> function meowbox(meow = 'meow') { document.getElementById("meowbox").innerHTML = meow; } </script>

The <script> tag is used to define this kind of client-side script.

There is also a <noscript> tag if you want to write HTML that will only appear if JavaScript is not enabled.

In our case, the JavaScript defined a function called meowbox(). This function will replace the text inside the following HTML tag:

<div id="meowbox">...</div>

Pretty simple, anytime meowbox() is called, it will replace the text in that <div>


Run it yourself

Type the following into the address bar above:

javascript:meowbox("this is from the address bar");

As a result, you can put JavaScript in a link as follows:

<a href="javascript:alert('This is an alert');">Click me</a>!

Click me!


But for testing it's easier to just use the console:

meowbox("hello from the console")

Most browsers will include a JavaScript console, see if your browser has one and run the above code. You should see the text underneath the kittens change to whatever you want.


Update the HTML

There are many ways to get JavaScript to work with your HTML, take a look:

<div id="kittenbox"> <img src="img_kittens_308x233.jpg" onclick="meowbox('meow!')" /> <img src="img_kittens_302x233.jpg" onclick="meowbox('meow meow meow meow')" /> <img src="img_kittens_350x233.jpg" onclick="meowbox('meow meow')" /> <img src="img_kittens_260x233.jpg" onclick="meowbox('meow?')" /> </div> <div id="meowbox">...</div>

The onclick attribute allows for you to write any JavaScript, and it will be executed when the user clicks on that element.

Pretty cool!


Hover and Mouse Events

In most cases, you can just use CSS for mouseover effects, for example:

#kittenbox img { float: left; padding: 4px; opacity: 0.4; filter: alpha(opacity=40); /* msie */ } #kittenbox img:hover { background-color: #ccc; opacity: 1.0; filter: alpha(opacity=100); /* msie */ }

The img:hover defines the style for when the mouse is hovering over that element.

* in this case I changed the opacity, which requires a small hack to work on older versions of Internet Explorer.


You can also use an onmouseover attribute:

<div id="meowbox">...</div> <div id="kittenbox"> <img src="img_kittens_308x233.jpg" onmouseover="meowbox('meow!')" /> <img src="img_kittens_302x233.jpg" onmouseover="meowbox('meow meow meow meow')" /> <img src="img_kittens_350x233.jpg" onmouseover="meowbox('meow meow')" /> <img src="img_kittens_260x233.jpg" onmouseover="meowbox('meow?')" /> </div>

Save this along with the above JavaScript in your example.html and see if you can get it working.


Event Handlers

Attributes like onclick and onmouseover are called Event Handlers, or also HTML DOM Events, and there are many of them.

Try playing around with other events, like ondblclick, ondrag, oncopy, etc.

How do we do this?

It's easy, the HTML code for the above kittens looks like this:

<div id="kittenbox"> <img src="img_kittens_350x233.jpg" /> <img src="img_kittens_260x233.jpg" /> <img src="img_kittens_302x233.jpg" /> <img src="img_kittens_308x233.jpg" /> </div> <br />

Let's try it yourself

Copy and paste the following HTML and make a new example.html

<!DOCTYPE html> <html> <head> <link href="style.css" rel="stylesheet" type="text/css" /> <meta charset="utf-8"> <title>Kittens!</title> </head> <body> Hello Kittens! <div id="kittenbox"> <img src="http://avant.net/lessons/img_kittens_350x233.jpg" /> <img src="http://avant.net/lessons/img_kittens_260x233.jpg" /> <img src="http://avant.net/lessons/img_kittens_302x233.jpg" /> <img src="http://avant.net/lessons/img_kittens_308x233.jpg" /> </div> <br /> </body> </html>

Notice the image src is set to the full URL of the image.

I did this so you don't have to download the images.


It doesn't look the same

If you opened your new example.html you'll notice that the images aren't aligned the way they are above.

This is because you need to add CSS to make things looks beautiful.

In particular, look at this line in the HTML

<div id="kittenbox">

You've already learned about the CSS class attribute, you can also use an id attribute.

The id is used for a specific element in the HTML, in this case, kittenbox contains the kitten images.

For reference, <div> tags are used for what is known as block elements, like images or a block of text.

And <span> is used for inline elements, like inside a line of text.

In practice, this just means <div> includes a line break, and <span> does not.


style.css

In order to get your CSS to refer to kittenbox, you'll use the follows CSS:

#kittenbox { width: 640px; }

If you save this in your style.css then your kitten images will already look much better.

But don't stop here, try some additional CSS and see what you can do!

Change the order of the images, align them in different ways, try this and see what happens:

#kittenbox { width: 1240px; text-align: center; } #kittenbox img { float: left; padding: 0px; margin: 0px; border: 0px; border-bottom: 10px solid white; } #kittenbox img:hover { border-bottom: 10px solid #ccc; }

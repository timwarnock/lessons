No magic, it's just HTML + CSS + Javascript (of course)

The first thing you do is create invisible elements on the page, like so:

<div id="header" style="display:none;"> ... </div>

You can also use an HTML5 header element, like so:

<header style="display:none;"> ... </header>

I used the div approach, and my CSS looks like this:

html, body { height: 100%; margin: 0; padding: 0; font-family: Sans-Serif; } #body { width: 100%; padding: 10px; } #header { width: 100%; border-bottom: 3px solid black; margin-bottom: 20px; padding: 20px 60px; font-size: 32pt; background-color: #333; color: #d0d0d9; }

Notice that I removed the margin and padding at the top-level of html and body.

I did this because I didn't want my header to have whitespace around it.

In order to make the rest of the page have the same margin it did before, I added a div#body.

The structure of the HTML is like this:

<html> <head>...</head> <body> <div id="header" style="display:none"> ... </div> <div id="body"> ... ... ... </div> </body> </html>

So all this really does is create a webpage that has a header that is not displayed, and looks exactly like it did before.

Not very useful on the surface, but this is where the Javascript comes in.

I created the following Javascript functions:

<script> function toggled() { for (var i=0; i < arguments.length; i++) { var el = document.getElementById(arguments[i]); el.style.display = el.style.display === 'none' ? '' : 'none'; } } function toggleindent() { var el = document.getElementById('body'); el.style.paddingLeft = el.style.paddingLeft === '60px' ? '10px' : '60px'; } </script>

These functions allow me to toggle the display of a given HTML element, as well as toggle the indent of the body.

All I needed to do was call the function in a link, like so:

<a href="javascript:toggled('readyformore');">like so</a>
